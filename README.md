# RedosStringBuilder

  Builds specially crafted strings for ReDoS attacks.

  [![Build Status][travis-image]][travis-url]

Install
-------
stable
```bash
npm install -g redos-string-builder
```
edge
```bash
npm install -g https://github.com/daxxog/redos-string-builder/tarball/master
```
